3D Mesh Clustering for Siemens

Title

Laplace-Beltrami-based clustering workflow for industrial 3D mesh analysis.

Objective

Create a scalable clustering methodology to improve part analysis and support labeled dataset generation.

Process

Engineered a workflow with Laplace-Beltrami eigenvalues, offset surfaces for manifold geometries, and BIRCH clustering to improve dendrogram structure and fidelity across 200,000 parts.

Tools

Simcenter STAR-CCM+, BIRCH clustering, Python-based ML workflows.

Value Proposition

Shows the ability to translate advanced ML concepts into practical industrial outcomes with high clustering accuracy and lower computational cost.
