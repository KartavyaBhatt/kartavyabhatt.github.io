AWS Script Runner Modernization

Title

Modernization of the AWS Script Runner experience and platform services.

Objective

Improve UI consistency, system performance, and scalability while supporting growing user demand.

Process

Upgraded frontend from Polaris 2.0 to 3.0, redesigned execution-result APIs, built GraphQL endpoints, optimized PostgreSQL queries, and migrated infrastructure to AWS CDK.

Tools

React, GraphQL, PostgreSQL, AWS, AWS CDK.

Value Proposition

Demonstrates end-to-end ownership across frontend, backend, and infrastructure with measurable impact on performance, satisfaction, and scalability.
