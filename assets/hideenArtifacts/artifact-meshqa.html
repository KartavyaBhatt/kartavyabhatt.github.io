Computer Vision

3D Mesh Quality Assurance Tool

C++ Python PyTorch Geometric PyQt OpenGL

The Objective

To fully automate the tedious manual quality inspection phases of industrial-grade 3D meshes before they are passed downstream into the standard rendering pipeline, solving a severe bottleneck in production.

The Process

Conducted extensive literature reviews on emerging geometric deep learning techniques. Implemented a Graph Neural Network (GNN) model leveraging this research, designed to natively ingest non-Euclidean data and classify localized topological errors (e.g., non-manifold edges, holes). Developed an intuitive PyQt frontend that allows QA operators to visually navigate directly to the flagged vertices on the 3D model.

Value Proposition

Drastically reduced manual inspection times by over 80%. Highlights specific competency in intersecting traditional computer graphics paradigms with bleeding-edge machine learning research to deliver immense automation value.
