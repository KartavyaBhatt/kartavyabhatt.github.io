Machine Learning

Applied Deepfake Detection System

PyTorch OpenCV FastAPI Python AWS EC2

The Objective

To engineer a highly accurate, real-time pipeline capable of classifying AI-generated video manipulations. The motive was to build a system that detects spatial and temporal inconsistencies frequently left by GAN-based synthetic media generators.

The Process

Aggregated a large-scale dataset of forged media spanning diverse conditions. Engineered a hybrid CNN-RNN architecture to capture both frame-level artifacts and sequence-based anomalies. Trained the model over multiple epochs utilizing AWS EC2 GPU instances, and wrapped the optimized inference logic into a highly concurrent FastAPI module for low-latency scoring.

Value Proposition

Demonstrates the capability to bridge the gap between abstract research-grade deep learning concepts and functionally deployable, high-throughput microservices. Highlights advanced competency in computer vision and scalable API deployment.
